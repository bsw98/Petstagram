## 🚀 프로젝트 사용법

```sh
npm i

npm start
```

**root경로에 .env파일이 필요합니다.**

```sh
# /.env
REACT_APP_API_URL=[API_URL]
```


## 🦁 Petstagram
### 본인의 애완동물사진을 공유하고 애완동물관련 내용을 공유하는 sns
<br>

## 💻 프로젝트 소개

애완동물 사진을 공유하고 사람들과 소통하는 sns입니다. <br>
애완동물 케어 서비스로 사용자들이 서로 애완동물을 돌봐줄 수 있는 기능도 제공합니다.
